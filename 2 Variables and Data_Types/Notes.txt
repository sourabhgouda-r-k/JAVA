What are variables?
 ->variables are containers used to store data in memory. 
 ->Variables define how data is stored, accessed, and manipulated.


Data Types: Data Type is an attribute associated with a piece of 
data that tells a computer system how to interpret its value.

 Type       Example       Meaning          
----------------------------------------     
 int        10            Integer number   
 double     10.5          Decimal number   
 char       'A'           Single character 
 String     "Hello"       Text             
 boolean    true/false    Logic value
